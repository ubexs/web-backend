# ubexs Web Backend Config
All ubexs configuration data is stored in a file called `config.json` (or, if using `ubexs/deploy`, then `config/config.web-backend.json` for production data).

This file will go over the general configuration

## Config
```json5
{
    // ALl MySQL DB data
    "mysql": {
        "host": "127.0.0.1",
        "user": "username",
        "password": "password",
        "database": "database_name",
        "charset": "utf8"
    },

    // In development, I edited my windows hosts file to forward *.blockshub.hh to localhost, however, if you don't want to do that then you can just use localhost
    "baseUrl": {

        // For example: https://api.ubexs.com
        "backend": "http://api.ubexs.local:3000",
        // Frontend (blockshub/web-frontend) API Proxy Base Url
        // For example: https://api.ubexs.com in production, or localhost:3000 in development
        "frontend": "http://api.ubexs.local:3000",
        // Game Client (blockshub/game-client) BASE url
        // For example: https://play.ubexs.com
        "play": "http://play.ubexs.local:3006",
        // WWW Base Url
        // For example: https://www.ubexs.com
        "www": "http://www.ubexs.local:3005"

    // If MySQL TLS is enabled
    "mysqlTLS": {
        "enabled": false

        "host": "127.0.0.1"
```
